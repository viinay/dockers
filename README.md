## Various Commands in Dockerfile

1] FROM - used to define the base image, on which we will be building

  `FROM image-tag` usually the 1st line of a Dockerfile
  
  Example:
    `FROM ubuntu:18.04` OR FROM `python:latest`
    
2] ADD

3] RUN

4] CMD

5] ENTRYPOINT

6] ENV

7] WORKDIR

8] COPY
